    • dsRNA Target Window

    Identifies an optimal window in a multiple alignment for dsRNA targeting

    Project maintained by Mitter-lab Hosted on GitHub Pages — Theme by mattgraham


    Developed by Stephen Fletcher @ the laboratory of Assoc. Prof. Neena Mitter, University of Queensland

    Required:


    Python 2.7
    R

    Installation


    • Install in your path
    • Make executable --> chmod +x path\to\ds_RNA_finder

    Usage:


    dsRNA_target_window input_alignment output_file [opt]


    Positional arguments:
    input_alignment multiple alignment in FASTA format
    output_file : output .csv file

    Optional arguments:
    -h : show this help message and exit
    -win_size : dsRNA window size
    -pos_bonus : bonus for complete conservation as a position
    -run_bonus : compounding bonus for a run of consered positions
